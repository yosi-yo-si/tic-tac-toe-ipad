# 施術フロータイマー (iPad向け)

リラクゼーション施術の進行を、設定から実行まで一貫して管理する Web アプリです。

## ファイル構成

- `index.html` - 画面構成（設定画面 / 実行画面）
- `style.css` - Apple風ミニマルUI、レスポンシブ、ダークモード対応
- `script.js` - 施術プラン編集、タイマー実行、LocalStorage保存

## 使い方

1. `index.html` をブラウザで開く
2. 基本設定（予約時間 / カウンセリング / 準備）を入力
3. 大枠メニューと部位ごとの工程を設定
4. テンプレート保存して再利用
5. 実行画面で開始・一時停止・スキップ・時間調整

## 備考

- 初回は `サンプル120分` テンプレートが自動登録されます。
- iPadの画面スリープ抑止はWeb単体では制限があるため、端末設定での対策を推奨します。
